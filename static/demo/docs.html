Your Documents, Tags, and Permissions

Plasmid is the only CMS that allows you to download any existing web page and use it as theme (including, images, css, js). It also provides a wide selection of themes.

After you choose a theme, Plasmid allows you to edit pages in place, tag them, and give permissions.

Plasmid uses group based access control. You can create groups, subscribe to groups, and give read/write permissions to any user or group for each page you make. Plamid takes care of user authentication and access control.

Plasmid allows you to embed many types of widgets into the pages, including comments, rating, audio/video players. It also allows you to include LaTeX formulas.
